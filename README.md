## Setup 
### Create project l
```php
composer create-project laravel/laravel project_name
```
### Setting Vue js
```php
composer require laravel/ui --dev
php artisan ui vue
npm install
```
### Run application
```php
php artisan serve
npm run dev
```
---
**NOTE**

npm run dev: Vite sẽ tự watch (theo dõi) các thay đổi trong code Vue và tự compile lại
